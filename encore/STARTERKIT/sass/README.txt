1. USAGE
Use a matching directory set and global SCSS file to dynamically build your SASS inclusions for compiling, 
or simply place your SCSS files in the global directory for direct inclusion. Use the theme's .info file 
to include the files in the 'global' directory. Each directory should include a global file preceded with
a double underscore (__) to be compiled before specific overrides in the directory. You will need to
manually control the order of 'layouts' to ensure your base layout is loading correclty.

2. EXAMPLE
Refer to 'blocks' for an example, and use this to copy and rename for other sections of the site you'd like
to control the styles and file structure for.

3. REQUIREMENTS
SASS - http://sass-lang.com/
COMPASS - http://compass-style.org/
SASS Globbing - https://github.com/chriseppstein/sass-globbing
